Password Generator - This was a challenge for creating a password generator app
using HTML, CSS and JavaScript. This challenge came with figma designs.

## View sites

- [Live Site](https://nolan-password-generator.netlify.app/)
- [Code View](https://github.com/nseokaneJHB/password-generator)

## Technologies

- **HTML**: Markup Language for creating a skeleton of a page.
- **CSS**: This is used to style the HTML page to make it look nice.
- **JavaScript**: A language used to make the page interactive (At least thats
  the basic of it's explanation).
- **Netlify**: Cloud platform to deploy websites.

## Features

- Generate a password based on the selected inclusion options
- Copy the generated password to the computer's clipboard
- See a strength rating for their generated password
- View the optimal layout for the interface depending on their device's screen
  size
- See hover and focus states for all interactive elements on the page
