Skate On – the only app that understands that your inner daredevil needs more
than just a skateboard! Whether you're shredding on a board, cruising on a
scooter, stunting with your bmx, or gracefully gliding on roller blades, we've
got the spots, events, and community to fuel your adrenaline addiction. Join us,
or you might just miss out on the fun – and we know how much you love to _not_
be left behind!

## Technologies

- **NextJs**: A React framework for building static and dynamic websites.
- **Next Auths**: A Library for implementing authentication simply by importing
  functions.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Vercel**: A cloud platform for static sites and serverless functions.
- **Prisma**: A cloud platform for a range of functionalities but used mainly
  for storing data.
- **Edge Store**: A cloud platform for storing images.
- **React Places Autocomplete**: A library for doing Google places addresses.
- **Google Generative AI**: A library for generating content using AI.
- **Resend**: A package for sending emails.
- **Tanstack React Query**: A library for easy data fetching and mutations.
- **React Email**: A library for easy of generating email templates.
- **Nuqs**: A library for implementing filters using url params.
- **React Hook Form**: A library for form state management.

## Features

- **Home**:
  - View Intro
  - View Events
  - View Spots
  - View Reviews
  - Subscribe To Skate On
- **Authentication**:
  - Magic link
  - Google Authentication
- **Profile**:
  - View Profile
  - Update Profile
  - Update Sponsors
- **Notifications**:
  - View Notifications
  - Filter Notifications
  - Delete Notifications (Single and Multiple)
  - Mark Notifications as Read or Unread (Single and Multiple)
- **Events**:
  - Add Event
  - View Events (All and Individual)
  - Search Events
  - Filter Events
  - Update Event
  - Delete Event
  - Event Location (Opens Google Maps)
  - Add Comment On Event
  - Like or Unlike Comment
- **Spots**:
  - Add Spot
  - View Spots (All and Individual)
  - Search Spots
  - Filter Spots
  - Rate Community
  - Like or Unlike Community
  - Update Spot
  - Delete Spot
  - Spot Location (Opens Google Maps)
  - Add Comment On Spot
  - Add Comment On Event
  - Like or Unlike Comment
- **Community**:
  - View Community (All and Individual)
  - Search Community
  - Filter Community
  - Rate Community
  - Like or Unlike Community
- **Account**:
  - Delete Account
- **Preference**:
  - View Account Subscriptions
  - Update Account Subscriptions
- **Reviews**:
  - Add Reviews
  - View Reviews
